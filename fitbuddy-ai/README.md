# FitBuddy AI – Personalized Fitness Assistant (In Progress)

## Overview
FitBuddy AI is a full-stack application focused on delivering personalized workout and nutrition guidance using AI-driven recommendations.

The goal of this project is to understand how real-world systems handle:
- User input
- API communication
- Data modeling
- Deployment workflows

## What I’m building
- A frontend interface to collect user fitness goals
- Backend APIs to process requests and manage user data
- AI-powered recommendations using external APIs
- Clean request/response flows between frontend and backend
